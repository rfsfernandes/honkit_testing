open override fun isFeatureSupported(featureType: FeatureType): BillingResponseCode

Checks if the specified feature type is supported for the given package.

Return

The corresponding BillingResponseCode indicating the result of the feature support check. If the billing service call is successful and the feature is supported, BillingResponseCode.OK is returned. If the billing service call is successful and the feature is NOT supported, BillingResponseCode.FEATURE_NOT_SUPPORTED. If the billing service is not available or a RemoteException is thrown, BillingResponseCode.SERVICE_UNAVAILABLE is returned. If an error occurs during the process, BillingResponseCode.ERROR is returned.

featureType

FeatureType The feature type to be validated.
